# XL-Sudoku-Solver

## Install
```
pip install xl-sudoku-solver
```

## Usage
Cmd:
```
problem1.txt:
9x37xxxx1
xxx6xx5xx
x4xxx1xxx
xxx36x2xx
8xxxxxxx4
xx6x97xxx
xxx5xxx2x
xx7xx3xxx
3xxxx28x9
```
```
$ xl-sudoku-solver --time -f problem1.txt
+-----------+-----------+-----------+
```
