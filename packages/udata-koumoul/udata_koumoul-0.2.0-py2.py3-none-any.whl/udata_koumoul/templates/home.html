Portail Open Data de démonstration

Ce portail est réalisé grâce à l'assemblage de différents logiciels Open Source

  • Le catalogue de données est assuré par uData, qui est également utilisé pour le site data.gouv.fr
  • DataFair, que nous développons, permet de rendre les données Faciles à Trouver, Accessible, Interopérables et Réutilisables (FAIR).
  • Les statistiques sont prises en charge par Matomo (anciennement Piwik)

N'hésitez pas à parcourir ce portail pour voir les différentes fonctionnalités offertes. Si vous êtes intéressés par la réalisation d'un portail, n'hésitez pas à nous contacter.
