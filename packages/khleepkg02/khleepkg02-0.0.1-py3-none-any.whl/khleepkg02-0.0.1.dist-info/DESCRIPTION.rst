# SSL-Lab-seminar-test

SSL-Seminar-test repository:: Topic - Package Test and Deployment

## Synopsis

```python
from ssllabtest import Caculator 

calc = Caculator(isAdd=True,
                 isSub=False,
                 isMul=False,
                 isDiv=False,
                 isRem=False)
sum = calc.adder(1, 2)
print(sum) # sum = 3
```
